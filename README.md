# Lim Hyejung Portfolio

Artist portfolio website for Lim Hyejung (임혜정), featuring her artworks and artist statement.

## About the Artist

Lim Hyejung (b.1981/Korea) explores nature reconstructed through imagination, presenting the perspective of "Utopia = Reality" where ideals and reality blur into living worlds.

## Features

- Full-screen hero section
- Chapter-based artwork organization
- Lightbox modal for artwork details
- Bilingual content (English/Korean)
- Reflective questions for each artwork
- Responsive design

## Tech Stack

- Next.js 14
- TypeScript
- Tailwind CSS
- React

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Project Structure

```
src/
├── app/          # Next.js app router
├── components/   # React components
├── data/         # JSON data (artist, artworks, chapters)
└── types/        # TypeScript types
```
